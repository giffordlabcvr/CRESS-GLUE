CRESS-GLUE

A GLUE project for circular Rep-encoding single-stranded DNA (CRESS DNA) viruses.

Download .zip Download .tar.gz View on GitHub

Installing GLUE

To utilise CRESS-GLUE via GLUE, you will first need to install the GLUE software framework. Note that installing GLUE requires that you install several other programs, including MySQL, which sometimes turns out to be challenging. To avoid this, you might want to consider a Docker-based installation

Loading the CRESS-GLUE project

On computers with GLUE installed, the CRESS-GLUE project can be instantiated by navigating to the project folder, initiating GLUE, and issuing the following command in the GLUE shell: 

  Mode path: /
  GLUE> run file cressProject.glue
