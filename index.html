CRESS-GLUE

A GLUE project for circular Rep-encoding single-stranded DNA (CRESS DNA) viruses.

Overview

This is CRESS-GLUE, a GLUE project designed to support comparative genomic and evolutionary analysis of circular Rep-encoding single-stranded DNA (CRESS DNA) viruses.

GLUE is a flexible software environment for working with molecular sequence data derived from viruses. GLUE projects are effectively mini-databases that capture information about specific viruses or virus groups. CRESS-GLUE focusses on CRESS DNA viruses, and aims to facilitate analysis of this poorly understood virus group.

What does it contain?

The CRESS-GLUE project contains data derived from representative CRESS-DNA virus species. For full details see here.
