How can it be used?

Like any GLUE resource, CRESS-GLUE is data-oriented, and can readily be developed in any direction. See here for some examples.

CRESS-GLUE can be used as a straightforward data repository (i.e. a place to find an annotated set of CRESS DNA virus genomes, information about CRESS DNA virus-derived EVEs, sequence alignments, and phylogenies.

However, the most effective way to leverage the information collated in the CRESS-GLUE project is to load it into the GLUE software framework, as described here.
