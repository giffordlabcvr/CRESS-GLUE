Nomenclature for ECVs

We applied a systematic approach to naming ECV, following a convention developed for endogenous retroviruses. Each element was assigned a unique identifier (ID) constructed from a defined set of components.

The first component is the classifier ‘ECV’.

The second is a composite of two distinct subcomponents separated by a period: (i) the name of ECV group; (ii) a numeric ID that uniquely identifies the insertion.

The third component of the ID defines the set of host species in which the ortholog occurs.

The numeric ID is an integer identifies a unique insertion locus that arose as a consequence of an initial germline infection. Thus, orthologous copies in different species are given the same number. Where an ECV lineage has been established via intragenomic amplication an incorporated ECV sequence, distinct duplicates are identified by numbers after the decimal point.
