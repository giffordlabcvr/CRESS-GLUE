Endogenous CRESS virus (ECV) data

We have used GLUE to organise the fossil record of CRESS viruses. You can access these data in this repository. This website provides quick links to specific data items.

ECV reference sequences and data

We reconstructed reference sequences for ECVs by aligning ECV sequences that derive from the same ancestral germline integration event.

For ECVs derived from circoviruses (genus Circovirus):
Tabular data;
Refererence FASTA nucleotide sequences (individual FASTA files);
Refererence FASTA protein sequences: Rep (aligned) (single file); Rep (unaligned) (individual FASTA files);

For ECVs derived from cycloviruses (genus Cyclovirus):

Tabular data.
Refererence FASTA nucleotide sequences (individual FASTA).
Refererence FASTA protein sequences (single file).

For ECVs derived from CRESS viruses:

Tabular data.
Refererence FASTA nucleotide sequences (individual FASTA).
Refererence FASTA protein sequences (single file) Rep (aligned). Rep (aligned).

Reference sequences were constructed by first constructing alignments of ECV sequences that are derived from the same initial germline colonisation event. These include orthologous elements in distinct species, and paralogous elements that have arisen via intragenomic duplication of ECV sequences (e.g. mediated by segmental duplication or transposition).

Raw ECV sequences and data

These are the raw data generated by database-integrated genome screening (DIGS). The tabular files contain information about the genomic location of each EVE. EVEs were classified by comparison to a polypeptide sequence reference library designed to represent the known diversity of CRESS viruses - this includes extinct lineages represented only by endogenous viral elements (EVEs).

The raw ECV locus data for circoviruses (genus Circovirus) and circovirus-like viruses are here. These data were obtained via DIGS performed in vertebrate genome assemblies downloaded from NCBI genomes (2020-07-15).

The raw ECV locus data for cycloviruses (genus Cyclovirus) are here. These data were obtained via DIGS performed in invertebrate genome assemblies obtained from NCBI genomes (2020-07-15).

The raw ECV locus data for CRESS viruses (genus Cyclovirus) are here. These data were obtained via DIGS performed in invertebrate genome assemblies obtained from NCBI genomes (2020-07-15).

Nomenclature for ECVs

We have applied a systematic approach to naming ECV, following a convention developed for endogenous retroviruses. Each element was assigned a unique identifier (ID) constructed from a defined set of components.

The first component is the classifier ‘ECV’ (endogenous CRESS virus element).

The second is a composite of two distinct subcomponents separated by a period: (i) the name of ECV group; (ii) a numeric ID that uniquely identifies the insertion.

The third component of the ID defines the set of host species in which the ortholog occurs. The numeric ID is an integer identifies a unique insertion locus that arose as a consequence of an initial germline infection. Thus, orthologous copies in different species are given the same number. Where an ECV lineage has been established via intragenomic amplication an incorporated ECV sequence, distinct duplicates are identified by numbers after the decimal point.
